# KPS Logistics

Sitio web corporativo para **KPS Logistics**, empresa chilena dedicada al transporte de carga por rutas troncales y transporte de personal corporativo.

## Estructura del Proyecto

```
kps-logistics/
  index.html          # Página principal
  css/styles.css      # Estilos del sitio
  js/main.js          # Interactividad y navegación
  assets/             # Recursos estáticos (imágenes futuras)
  README.md           # Este archivo
```

## Secciones

1. **Hero** - Presentación principal con llamado a la acción
2. **Servicios** - Transporte de Carga y Transporte de Personal
3. **Flota** - Camiones y Vans/Minibuses con especificaciones técnicas
4. **Licitaciones** - Capacidades documentales y de cumplimiento
5. **Sobre KPS** - KPS Group y sus divisiones
6. **Contacto** - Formulario completo + WhatsApp

## Tecnologías

- HTML5 semántico
- CSS3 (variables, grid, flexbox, responsive)
- JavaScript vanilla (ES5+ compatible)
- Google Fonts (Inter)
- Imágenes vía Unsplash

## Despliegue

Abrir `index.html` en cualquier navegador o servir con cualquier servidor estático. Compatible con GitHub Pages.

## Parte de KPS Group

KPS Logistics es la división operativa de **KPS Group**, plataforma empresarial orientada a transporte, logística y futuras divisiones (KPS Capital, KPS Leasing).
